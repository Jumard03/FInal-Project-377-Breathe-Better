# BreatheBetter NL

## Project Description

BreatheBetter NL is a web application that provides real-time air quality information for stations in the Netherlands. Users can search for stations, view PM2.5 and AQI data, and receive health recommendations based on current air quality.

## Target Browsers

- Chrome
- Firefox 
- Edge
- Safari 
- Mobile Chrome and Safari
